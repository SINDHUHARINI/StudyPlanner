# 📚 Study Planner

A simple and responsive **Study Planner Web Application** that helps students organize study schedules, manage tasks, and stay productive.

## 🚀 Features

* Plan daily study schedules
* Add and manage study tasks
* Clean and responsive UI
* Fast deployment using Vercel

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* React.js
* Vercel (Deployment)

---

## ⚙️ Run Locally

Clone the project



Go to the project directory

cd study-planner

Install dependencies

npm install

Start the server

npm start
